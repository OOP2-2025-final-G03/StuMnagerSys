# 最終課題
## 学生管理システム

### チームメンバー

- k24019
- k24100
- k24039 
- k24056 

### 使用するもの
- Flask
- SQLite

### データベース
- ユーザー情報（ID、名前、年齢、住所、電話番号、メールアドレス、設定情報、種類（学生/教師））
- パスワード情報（ID、パスワード）
- 成績情報（ID、学生ID、科目名、点数）
- 科目情報（ID、科目名、担当教師）

### 機能要件
- ログイン機能（学生、教師共通）
- ユーザー情報の登録、修正、削除、検索
- 成績情報の登録、修正、削除、検索
- 科目情報の登録、修正、削除、検索
- 成績情報の集計（平均点、最高点、最低点）
- 成績情報の可視化（グラフや表形式での表示）
- 未来の成績の予測機能
- 全体/学生ごとの成績分析(得意、苦手など)


### 役割分担
| 役割 | 担当 |
| --- | --- |
| ログイン機能 | チームリーダー |
| 成績情報および成績に関する機能 | チームメンバー |
| 科目情報および科目に関する機能 | チームメンバー |
| ユーザー情報およびユーザーに関する機能 | チームメンバー |
